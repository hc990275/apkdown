## 仓库简介

`apkdown.sh` 是一个在安卓环境（推荐 MT 管理器拓展包）运行的 Bash 脚本，自动检测多个 TVBox 相关渠道的版本信息并批量下载最新 APK。脚本会把版本文件与 APK 文件统一保存到 `/storage/emulated/0/0网站/下载专用/影视安装包更新` 目录下，方便对比与分发。完整的功能说明可查看《[软件更新脚本说明](SCRIPT_INTRODUCTION.md)》。

## 快速开始

1. 将仓库中的 `apkdown.sh` 拷贝到安卓设备，并在 **MT 管理器拓展包** 环境运行：
   ```bash
   bash apkdown.sh
   ```
2. 确保能够访问 GitHub 的 Raw 与 API 域名，脚本会自动创建下载目录与版本缓存目录。首次运行会拉取版本信息并下载对应 APK。
3. 脚本结束时会以彩色输出汇总报告，显示各渠道的旧/新版本号与下载结果，如需重试直接重新执行脚本即可。

## 核心功能速览

- **自我更新**：调用 GitHub Releases 检查脚本新版本，若发现更新则自动下载替换后提示重启脚本。
- **版本比对与缓存**：为 OK 版（手机/电视/Pro）与蜜蜂版（手机/电视）拉取 JSON 或文本版本号，记录旧版本与新版本并决定是否下载。
- **批量 APK 下载**：内置 32/64 位、海信专版、安卓 4 版本及 Pro 变体的映射表，检测到更新后自动按渠道批量下载。
- **执行结果汇总**：以随机彩色输出展示版本变化与下载状态，并附上公众号提示信息。
